# Norske Site

Vercel-ready Next.js project for the Norske Recreational Innovations landing page.

## Local development

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Create a new GitHub repository.
2. Upload all files from this folder to the repository.
3. In Vercel, import the GitHub repository.
4. Click Deploy.
5. In Vercel project settings, add your custom domain.
6. In GoDaddy DNS, point the domain to Vercel.

## Notes

- The current category cards use placeholder stock images. Replace them later with your own assets in `app/page.tsx`.
- Your established logo is already included in `public/logo.png`.
